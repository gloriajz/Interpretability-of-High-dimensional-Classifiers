Interpretability of High-Dimensional Classifiers

Gloria J. Yang ‚Äì UCL BSc Dissertation Project

üîç Overview

This project presents a comprehensive exploration of classification techniques and their interpretability within machine learning. We examine logistic regression, random forests, and neural networks across various datasets. The primary focus lies in understanding model decision-making through variable importance methods, Shapley values, and active subspace techniques.

üß™ Experiments

Three experiments were conducted to evaluate model interpretability:

  • Iris Dataset: Classifying species (Virginica vs Versicolor) and interpreting feature influence.
  • Simulated Datasets: Exploring synthetic data with both linear and non-linear structures to test sensitivity and variable interactions.
  • Credit Risk Dataset: Assessing the impact of borrower features on loan status prediction using interpretable models.

‚öôÔ∏è Methods

  • Variable Importance: Feature relevance quantified through z-scores, impurity reductions, and neural weight attributions.
  • Shapley Values: Feature contributions analyzed using cooperative game theory and implemented with fastshap.
  • Active Subspace: Gradient-based PCA for identifying the most influential directions in high-dimensional space.

üìä Results

Key findings across all experiments:

  • Iris: Petal length and petal width were the most discriminative features.
  • Simulation: Dataset 1 highlighted single-variable dominance; Dataset 2 revealed joint-variable interactions.
  • Credit Risk: Debt-to-income ratio, loan grade, and home ownership were the top predictive features for loan approval.
